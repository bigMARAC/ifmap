## Instalação

Na raiz use, isso irá instalar todas as dependencias que estão no `package.json`:
```bash
npm install
```

## Inicialização
Para inicializar o projeto, use:
```bash
npm run serve
```
## Requisitos
- [Node](https://nodejs.org/en/download/)
